自动化工具步骤

  • 初始化: npm init

  • 全局安装: npm install –global gulp

  • 作为项目的开发依赖（devDependencies）安装：npm install –save-dev gulp

    引入其他插件npm install –save-dev gulp-插件名称（已知less/jsmin/cssmin）

    例如：npm install gulp-jade –save-dev

  • 在项目根目录下创建一个名为 gulpfile.js 的文件：

  • 运行gulp


``
gulpfile.js文件

var gulp = require(‘gulp’);
var concat = require(‘gulp-concat’);
var jsmin = require(‘gulp-jsmin’);
var cssmin = require(‘gulp-cssmin’);

gulp.task(‘default’, function() {//可以有三个参数，[]
// 将你的默认的任务代码放在这
console.log(‘哈哈’)
});
gulp.task(‘hebing’, function() {//合并js文件

gulp.src(‘js/.js’)//进入js文件夹，（通配符）所有的后缀为js的文件
.pipe(concat(‘c.js’))//创建新德js，未定义，默认给你生成new.js
.pipe(gulp.dest(‘js/‘))//生成的new.js文件放在js文件里面
});
gulp.task(‘yashuo’,function(){压缩js文件
gulp.src(‘js/c.js’)//找到js文件
.pipe(jsmin())//js文件的名称
.pipe(gulp.dest(‘js/‘))//压缩js文件放在的地方

});

```
