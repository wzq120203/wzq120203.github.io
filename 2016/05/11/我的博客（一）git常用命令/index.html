写博客/更新博客步骤： 

git常用命令
    git clone
        下载线上仓库地址
    git status
        查看当前的仓库状态
    git add
    git commit
        提交到暂存区
    git remote add origin git@github.com:yourname/learngit.git
    git push -u origin master 
本地配置
        git config —global user.name “Your Name”
        配置用户名
    git config —global user.email “email@example.com (mailto:email@example.com)”
        配置邮箱

完成之后更新博客四个步骤：

git add .

git commit -m ”更新博客 “
